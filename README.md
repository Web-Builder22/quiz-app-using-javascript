# 🚀 Programming Quiz App

An interactive web-based Programming Quiz Application that helps users test and improve their programming knowledge through multiple-choice quizzes.

The application provides topic-based quizzes including **C++ Programming, Java Programming, Web Development, and Data Structures & Algorithms** with instant feedback and score calculation.

---

## 📌 Features

✅ User name input system  
✅ Topic-based quiz selection  
✅ Multiple-choice questions  
✅ Instant correct/wrong answer feedback  
✅ Score calculation  
✅ Dynamic question rendering  
✅ Restart quiz option  
✅ Simple and user-friendly interface
